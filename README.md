# Jogo da Adivinhação


- Aplicativo Android de adivinhar um número feito com Java
- A máquina e o usuário escolhem um número de 1 a 3
- Em caso de erro ou acerto é mostrada uma mensagem na tela
- Tecnologia utilizada:
  - Java
